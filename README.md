# NUMIE

> A scalable multi-function group robot for QQ.

## USAGE

1. Edit [config.default.js](config.default.js)

2. Run docker based on [Dockerfile](Dockerfile)

3.

```bash
# Installation Dependencies
yarn install 

# Run numie on dev environment
yarn dev
```

## Requirements

- Docker

- Node.js
